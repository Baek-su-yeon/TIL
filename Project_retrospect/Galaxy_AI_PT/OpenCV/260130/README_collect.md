# 🖐️ 제스처 데이터 수집 가이드 (Data Collection Guide)

이 문서는 모델 고도화를 위한 데이터 수집 절차와 규칙을 설명합니다. 

## 1. 환경 설정 (Setup)

터미널(또는 Anaconda Prompt)을 열고 아래 명령어를 순서대로 입력하여 개발 환경을 구축합니다.

### **가상환경 생성**

```bash
# 가상환경 이름은 마음대로 생성 (Python 3.11 권장)
conda create -n YOUR_ENV_NAME python=3.11 -y

```

### **가상환경 활성화**

```bash
conda activate YOUR_ENV_NAME

```

### **필수 라이브러리 설치**

```bash
# 프로젝트 루트 디렉토리에서 실행
pip install -r requirements.txt

```

---

## 2. 데이터 수집 법칙 (Collection Rules)

1차 학습 결과 분석을 토대로 마련된 **데이터 수집 5계명**입니다.

### **① 9등분 화면 분할 수집 (Seated & Standing)**

* **앉은 자세**: 카메라 정면, 몸 고정 후 화면을 9등분(좌상단 ~ 우하단)하여 각 위치에서 촬영합니다.
* **선 자세**: 몸 전체가 나오는 거리에서 몸 고정 후 동일하게 9등분하여 촬영합니다.
* **Sweep 기법**: 좌상단에서 우하단까지 궤적을 그리며 훑는 방식으로 연속 데이터를 확보합니다.

### **② 다각도 촬영 (Palm/Side/Back)**

* 단순한 손바닥 정면뿐만 아니라 **손날(Side)**, **손등(Back)** 방향의 데이터를 모두 포함해야 합니다.


### **③ MediaPipe 랜드마크 확인**

* 화면에 초록색 랜드마크가 정확히 그려지는지 확인하고 저장하세요.
* 손이 화면 밖으로 나가거나 랜드마크가 뭉개진 데이터는 수집하지 않습니다.

### **④ 예외 케이스(None) 집중 수집 (지정된 인원 전용)**

* 1차 모델을 가동하여 **제스처가 아닌데 제스처로 오인식하는 순간**을 포착하여 `None` 데이터로 저장합니다. 


* 손가락을 접는 도중 발생하는 모호한 구간을 집중적으로 촬영하세요. 



### **⑤ 거리의 적절성**

* 랜드마크 정규화 덕분에 아주 멀리서 찍을 필요는 없으나, 실제 발표 거리(약 1.5~3m) 데이터를 포함합니다.

## 3. 실행 및 조작 방법 (How to Run)

수집할 제스처 이름을 설정한 후 스크립트를 실행합니다.
```python
# image_collect.py 파일에서 아래 코드 괄호 안에 이름을 바꾸면됨
# 협의한 클래스명 그대로 적을 것
collector.set_gesture("None")
```

### **실행 명령어**

```bash
# 최상단 폴더에서 입력
python -m image_collect

```
