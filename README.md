## 📂 폴더 구조 및 컨벤션

* **Naming Convention**: 변수 및 폴더명은 목적에 따라 camelCase, PascalCase, SNAKE_CASE를 엄격히 구분하여 사용합니다.
* **Documentation**: 모든 학습 내용은 핵심 개념 정리와 실습 코드를 포함하며, 프로젝트의 경우 설계도와 개발 회고를 동반합니다.
