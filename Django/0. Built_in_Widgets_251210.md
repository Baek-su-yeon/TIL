# Django Built-in Widgets 정리

Django Form 필드는 기본적으로 적절한 HTML 위젯을 가지고 있지만, `widget=` 옵션을 통해 렌더링 방식을 변경할 수 있습니다.

| 분류 | 위젯 클래스명 (Widget) | 렌더링되는 HTML 태그 | 설명 |
| :--- | :--- | :--- | :--- |
| **텍스트 입력** | `TextInput` | `<input type="text">` | 가장 기본적인 한 줄 텍스트 입력 |
| | `NumberInput` | `<input type="number">` | 숫자 입력 (HTML5) |
| | `EmailInput` | `<input type="email">` | 이메일 입력 (HTML5 형식 검사) |
| | `URLInput` | `<input type="url">` | URL 주소 입력 (HTML5) |
| | `PasswordInput` | `<input type="password">` | 비밀번호 입력 (입력 내용 가림) |
| | `HiddenInput` | `<input type="hidden">` | 화면에 보이지 않지만 데이터 전송 필요 시 사용 |
| **날짜/시간** | `DateInput` | `<input type="text">` | 날짜 입력 (JS 달력 연동 시 자주 사용) |
| | `DateTimeInput` | `<input type="text">` | 날짜 + 시간 입력 |
| | `TimeInput` | `<input type="text">` | 시간 입력 |
| **긴 텍스트** | `Textarea` | `<textarea>...</textarea>` | 여러 줄의 텍스트 입력 (게시글 본문 등) |
| **선택 (Boolean)** | `CheckboxInput` | `<input type="checkbox">` | True/False 체크박스 |
| **선택 (Select)** | `Select` | `<select><option>...</select>` | 드롭다운 목록 (단일 선택) |
| | `NullBooleanSelect` | `<select>` | 알 수 없음(Unknown)/예/아니오 3가지 선택 |
| | `SelectMultiple` | `<select multiple>` | 드롭다운 목록 (다중 선택, Ctrl/Cmd 클릭) |
| | `RadioSelect` | `<input type="radio">` 목록 | 라디오 버튼 리스트 (단일 선택) |
| | `CheckboxSelectMultiple` | `<input type="checkbox">` 목록 | 체크박스 리스트 (다중 선택) |
| **파일 업로드** | `FileInput` | `<input type="file">` | 파일 업로드 |
| | `ClearableFileInput` | `<input type="file">` + 체크박스 | 이미지 수정/삭제 기능이 포함된 업로드 위젯 |

참고: https://docs.djangoproject.com/ko/4.2/ref/forms/widgets/#built-in-widgets
