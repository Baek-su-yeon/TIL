### C문제 - 예제 1.c

```c
#include <stdio.h>
main() {
	int score[] = { 86, 53, 95, 76, 61 };
	char grade;
	char str[] = "Rank";
	for (int i = 0; i < 5; i++) {
		switch (score[i] / 10) {
		case 10:
		case 9:
			grade = 'A';
			break;
		case 8:
			grade = 'B';
			break;
		case 7:
			grade = 'C';
			break;
		default: grade = 'F';
		}
		if (grade != 'F')
			printf( "%d is %c %s\n", i + 1, grade, str);
	}
}
```

```c
// 예상 정답
1 is B Rank
2 is F Rank
3 is A Rank
4 is C Rank
5 is F Rank

// 실제 정답
1 is B Rank
3 is A Rank
4 is C Rank
```

- for문은 score 배열을 순차적으로 순회하며 각 요소의 몫에 대한 case를 실행할 것
- 각 case마다 break가 있으므로 case 10 외에 나머지 case는 실행 후 바로 if문으로 넘어감
- 53과 61은 해당하는 case가 없으므로 default 값인 F가 출력될 것
- 예외 사항으로 case 10이 선택되는 경우에 case 10에서 실행되는 것은 없지만 break가 없어서  case 9로 넘어간 후 grade에는 A가 할당될 것임

[오답풀이-gpt]

- if (grade != 'F') 조건 때문에 F 출력안함
- 코드 끝까지 잘 읽을 것

### 기출 따라잡기 - 문제 3.c

```c
#include <stdio.h>
#include <stdlib.h>
main(int argc, char *argv[]) {
	int v1 = 0;
	int v2 = 35;
	int v3 = 29;
	if (v1 > v2 ? v2 : v1)
		v2 = v2 << 2;
	else
		v3 = v3 << 2;
	printf("%d", v2 + v3);
	return 0;
}
```

```c
// 예상 정답
55 // 줄바꿈 없음

// 진짜 정답
151
```

- v1 > v2 ?는 0 > 35로 거짓이므로 if(0)이 됨
- else문으로 넘어가서 v3 = v3 << 2 = 29 << 2 = 11101 << 2 = 10100 = 16 + 4 = 20
- v2 + v3 = 35 + 20 = 55

[오답풀이-gpt]

- 와 이것도 바보짓했네
- 11101 << 2 → 1110100임 즉 정수 기준 $2^2$ 배 하겠다는 것,,,
- 1110100 = 64 + 32 + 16 + 4 = 116 (29 * 4 = 116)
- 35 + 116 = 151

### 03 Java 문제 - 예제 1.java

```java
public class Test {
	public static void main(String[] args) {
		String str = "agile";
		int x[] = { 1, 2, 3, 4, 5 };
		char y[] = new char[5];
		int i = 0;
		while (i < str.length()) {
			y[i] = str.charAt(i);
			i++;
		}
		for (int p : x) {
			i--;
			System.out.print(y[i]);
			System.out.print(p + " ");
		}
	}
}
```

```c
// 예상 정답
e1 l2 i3 g4 a5 
```

- while문은 str 길이인 5만큼 반복
- 그리고 y = {a, g, i, l, e}로 while 안에서 새로 할당될 것 같음
- for 문은 python 처럼 x 배열을 순환하겠다는 것 같고, p에는 x 배열의 요소들이 순차적으로 할당될 듯
- i++과 i—로 인해 y는 거꾸로 출력될 것임

```java
public class Test {
```

```java
			result[i] = 1;

	}
}
```

```java
// 예상 정답
23412
```

- result[0] = 1 → 2
- result[1] = 1 → 2 → 3
- result[2] = 1 → 2 → 3 → 4
- result[3] = 1
- result[4] = 1 → 2
- 줄바꿈 없어서 일렬로 쭉 출력

[오답풀이-gpt]

- 이 코드는 배열 값의 **내림차순 순위**를 구하는 코드야. 자기보다 큰 값이 있을 때마다 순위가 1씩 밀려.
    
    77 → 자기보다 큰 값: 99 → 순위 2
    32 → 자기보다 큰 값: 77, 99, 50 → 순위 4
    10 → 자기보다 큰 값: 77, 32, 99, 50 → 순위 5
    99 → 자기보다 큰 값 없음 → 순위 1
    50 → 자기보다 큰 값: 77, 99 → 순위 3
