## 1. Built-in Template Filters (주요 필터 정리)
*출처: Django Project Official Documentation (Built-in template tags and filters)*

| 필터명 | 설명 | 예시 |
| :--- | :--- | :--- |
| **add** | 값에 인자를 더함 (숫자는 덧셈, 문자는 연결) | `{{ value\|add:"2" }}` |
| **addslashes** | 따옴표 앞에 슬래시(`\`)를 추가 | `{{ value\|addslashes }}` |
| **capfirst** | 첫 글자만 대문자로 변환 | `{{ value\|capfirst }}` |
| **center** | 주어진 너비 내에서 값을 중앙 정렬 | `{{ value\|center:"15" }}` |
| **cut** | 특정 문자열을 모두 제거 | `{{ value\|cut:" " }}` (공백 제거) |
| **date** | 날짜를 지정된 형식으로 포맷팅 (자주 사용) | `{{ value\|date:"Y-m-d" }}` |
| **default** | 값이 False(None, 빈 문자열 등)일 때 기본값 출력 | `{{ value\|default:"nothing" }}` |
| **default_if_none** | 값이 오직 None일 때만 기본값 출력 | `{{ value\|default_if_none:"nothing" }}` |
| **dictsort** | 딕셔너리 리스트를 특정 키 기준으로 정렬 | `{{ value\|dictsort:"name" }}` |
| **dictsortreversed** | `dictsort`의 역순 정렬 | `{{ value\|dictsortreversed:"name" }}` |
| **divisibleby** | 값으로 나누어 떨어지면 True 반환 | `{{ value\|divisibleby:"3" }}` |
| **escape** | HTML 특수 문자를 이스케이프 처리 (`<` → `&lt;`) | `{{ value\|escape }}` |
| **escapejs** | 자바스크립트 문자열에 사용할 수 있도록 이스케이프 | `{{ value\|escapejs }}` |
| **filesizeformat** | 숫자를 파일 크기 형식(KB, MB)으로 변환 | `{{ value\|filesizeformat }}` |
| **first** | 리스트의 첫 번째 항목 반환 | `{{ value\|first }}` |
| **floatformat** | 부동 소수점 숫자를 지정된 자릿수로 반올림 | `{{ value\|floatformat:2 }}` |
| **force_escape** | HTML 이스케이프를 강제 적용 (즉시 적용) | `{{ value\|force_escape }}` |
| **get_digit** | 끝에서부터 n번째 자릿수를 가져옴 | `{{ value\|get_digit:"2" }}` |
| **iriencode** | IRI를 URI로 변환 (국제화 도메인 등) | `{{ value\|iriencode }}` |
| **join** | 리스트를 특정 문자열로 연결 (Python의 join과 동일) | `{{ value\|join:" // " }}` |
| **json_script** | Python 객체를 JSON으로 변환하여 `<script>` 태그로 출력 | `{{ value\|json_script:"hello-data" }}` |
| **last** | 리스트의 마지막 항목 반환 | `{{ value\|last }}` |
| **length** | 값의 길이(문자열 길이 또는 리스트 항목 수) 반환 | `{{ value\|length }}` |
| **length_is** | 길이가 특정 값과 같은지 확인 (True/False) | `{{ value\|length_is:"4" }}` |
| **linebreaks** | 줄바꿈(`\n`)을 `<p>`나 `<br>` 태그로 변환 | `{{ value\|linebreaks }}` |
| **linebreaksbr** | 줄바꿈을 오직 `<br>` 태그로만 변환 | `{{ value\|linebreaksbr }}` |
| **linenumbers** | 텍스트에 줄 번호를 표시 | `{{ value\|linenumbers }}` |
| **ljust** | 주어진 너비 내에서 왼쪽 정렬 | `{{ value\|ljust:"10" }}` |
| **lower** | 모든 문자를 소문자로 변환 | `{{ value\|lower }}` |
| **make_list** | 문자열이나 숫자를 리스트로 변환 | `{{ value\|make_list }}` |
| **phone2numeric** | 전화번호의 문자를 숫자로 변환 (예: 1-800-COLLECT) | `{{ value\|phone2numeric }}` |
| **pluralize** | 숫자에 따라 복수형 접미사('s' 등) 반환 | `{{ num\|pluralize }}` 또는 `{{ num\|pluralize:"y,ies" }}` |
| **pprint** | 디버깅용으로 객체를 예쁘게 출력 (pprint 모듈) | `{{ value\|pprint }}` |
| **random** | 리스트에서 무작위 항목 하나 반환 | `{{ value\|random }}` |
| **rjust** | 주어진 너비 내에서 오른쪽 정렬 | `{{ value\|rjust:"10" }}` |
| **safe** | HTML 이스케이프를 하지 않음 (보안 주의) | `{{ value\|safe }}` |
| **safeseq** | 리스트의 각 요소에 `safe` 필터 적용 | `{{ list\|safeseq }}` |
| **slice** | 리스트나 문자열을 슬라이싱 (Python 슬라이싱과 동일) | `{{ value\|slice:":2" }}` |
| **slugify** | 공백을 하이픈으로, 특수문자 제거 (URL용) | `{{ value\|slugify }}` |
| **stringformat** | 문자열 포맷팅 (Python의 `%` 연산자 방식) | `{{ value\|stringformat:"E" }}` |
| **striptags** | 모든 HTML 태그 제거 | `{{ value\|striptags }}` |
| **time** | 시간을 지정된 형식으로 포맷팅 | `{{ value\|time:"H:i" }}` |
| **timesince** | 기준 시간으로부터 얼마가 지났는지 표시 | `{{ blog_date\|timesince:comment_date }}` |
| **timeuntil** | 기준 시간까지 얼마가 남았는지 표시 | `{{ conference_date\|timeuntil:from_date }}` |
| **title** | 각 단어의 첫 글자를 대문자로 (Title Case) | `{{ value\|title }}` |
| **truncatechars** | 지정된 글자 수까지만 표시하고 말줄임표(...) | `{{ value\|truncatechars:9 }}` |
| **truncatechars_html** | HTML 태그를 인식하며 글자 수 자르기 | `{{ value\|truncatechars_html:9 }}` |
| **truncatewords** | 지정된 단어 수까지만 표시하고 말줄임표(...) | `{{ value\|truncatewords:2 }}` |
| **truncatewords_html** | HTML 태그를 인식하며 단어 수 자르기 | `{{ value\|truncatewords_html:2 }}` |
| **unordered_list** | 리스트를 재귀적으로 `<ul>`, `<li>`로 변환 | `{{ list\|unordered_list }}` |
| **upper** | 모든 문자를 대문자로 변환 | `{{ value\|upper }}` |
| **urlencode** | URL에 사용할 수 있도록 문자열 인코딩 | `{{ value\|urlencode }}` |
| **urlize** | 텍스트 내의 URL을 클릭 가능한 링크로 변환 | `{{ value\|urlize }}` |
| **urlizetrunc** | `urlize`와 같으나 링크 텍스트 길이를 제한 | `{{ value\|urlizetrunc:15 }}` |
| **wordcount** | 단어의 개수를 반환 | `{{ value\|wordcount }}` |
| **wordwrap** | 지정된 글자 수마다 줄바꿈 | `{{ value\|wordwrap:5 }}` |
| **yesno** | True/False/None에 따라 지정된 문자열 매핑 | `{{ value\|yesno:"yeah,no,maybe" }}` |

## 2. Built-in Template Tags (주요 태그 정리)
*출처: Django Project Official Documentation (Built-in template tags and filters)*

| 태그명 | 설명 | 비고 |
| :--- | :--- | :--- |
| **autoescape** | 현재 블록의 HTML 자동 이스케이프 동작 제어 (on/off) | `{% autoescape off %}` ... `{% endautoescape %}` |
| **block** | 자식 템플릿에서 오버라이드할 수 있는 블록 정의 | 상속(`extends`) 시 필수 |
| **comment** | 템플릿 주석 처리 (브라우저 소스에도 안 보임) | `{% comment %}` ... `{% endcomment %}` |
| **csrf_token** | CSRF 공격 방지를 위한 토큰 생성 | Form 내부 필수 사용 |
| **cycle** | 반복문 내에서 문자열을 순환하며 출력 (배경색 교차 등) | `{% cycle 'row1' 'row2' %}` |
| **debug** | 디버그 정보(현재 컨텍스트, 모듈 등)를 출력 | 개발 모드에서 유용 |
| **extends** | 부모 템플릿을 상속받음을 명시 | 파일 최상단 위치 |
| **filter** | 특정 블록 전체에 필터를 적용 | `{% filter upper %}` ... `{% endfilter %}` |
| **firstof** | 나열된 변수 중 False가 아닌 첫 번째 값을 출력 | `{% firstof var1 var2 "fallback" %}` |
| **for** | 배열이나 리스트를 순회 (Loop) | `forloop` 카운터 변수 사용 가능 |
| **if** | 조건문 (True/False 판단) | `elif`, `else` 사용 가능 |
| **ifchanged** | 루프 내에서 값이 변경되었을 때만 내용 출력 | 날짜별 그룹핑 등에 사용 |
| **include** | 다른 템플릿 파일을 현재 위치에 로드하여 렌더링 | `{% include "nav.html" %}` |
| **load** | 커스텀 템플릿 태그/필터 라이브러리 로드 | `{% load static %}` |
| **lorem** | 랜덤한 로렘 입숨(Lorem Ipsum) 텍스트 생성 | 테스트용 텍스트 |
| **now** | 현재 날짜 및 시간을 출력 | `{% now "j S F Y H:i" %}` |
| **regroup** | 리스트를 특정 속성 기준으로 그룹화 | `{% regroup city_list by country as country_list %}` |
| **resetcycle** | `cycle` 태그의 순서를 초기화 | |
| **spaceless** | HTML 태그 사이의 공백(탭, 줄바꿈)을 제거 | `{% spaceless %}` ... `{% endspaceless %}` |
| **templatetag** | 템플릿 문법 문자(`{%`, `{{` 등)를 그대로 출력할 때 사용 | 이스케이프 용도 |
| **url** | URL 패턴 이름을 통해 절대 경로(URL)를 반환 | `{% url 'articles:index' %}` |
| **verbatim** | 템플릿 엔진이 해석하지 말아야 할 블록 지정 | JS 프레임워크와 혼용 시 사용 |
| **widthratio** | 비율 계산 (바 차트 등을 만들 때 사용) | `{% widthratio 100 200 100 %}` |
| **with** | 복잡한 변수에 간단한 이름을 별칭으로 부여 (캐싱 효과) | `{% with total=business.employees.count %}` |
