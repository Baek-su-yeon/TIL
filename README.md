# 📚 Today I Learned (TIL)

이 레포지토리는 **개인 학습 기록** 및 **지식 저장소**입니다. 컴퓨터 과학의 핵심 개념, 알고리즘, 자료구조, 그리고 기타 프로그래밍 관련 지식을 학습하고 정리한 내용을 담고 있습니다.

## 📌 레포지토리 구성

모든 학습 내용은 주제별 폴더로 구분되어 관리됩니다. 각 폴더는 해당 주제의 **핵심 개념 정리**와 함께 관련 **문제 풀이 코드**를 포함하고 있습니다.

| 폴더 | 주요 내용 | 목표 |
| :--- | :--- | :--- |
| `algorithm/` | **자료구조(Stack, List), 정렬/탐색 알고리즘, DP, 재귀** 등 | 기본기 확보 및 코딩 테스트 대비 |
| `[추후 추가]` | 새로운 학습 주제 (예: Network, OS, Database 등) | 학습 영역 확장 및 지식 심화 |
